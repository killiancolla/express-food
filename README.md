## Download

```sh
git clone https://github.com/killiancolla/express-food
cd express-food
```

## Installation

Create .env file with connection parameters:

```js
DATABASE_USER=
DATABASE_PASSWORD=
JWT_SECRET=
PORT=
```

### Back-end

```sh
cd server
npm install
npm run server
```

### Front-end

```sh
cd express-food-front
npm install
npm start
```
